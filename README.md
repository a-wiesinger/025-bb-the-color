# Level 24 - Boss Battle - "The Color"
Following along with the C# Player's Guide by RB Whitaker
## Criteria:
1. Define a Color class with properties for Red, Green, and Blue
2. Add appropriate constructors
3. Create static properties for easy access to 8 pre-defined colors
4. In main method - create 2 Color-typed variables. 
   - Make one an instance
   - Make one using a static property
5. Each of the above variables should display their RGB values
